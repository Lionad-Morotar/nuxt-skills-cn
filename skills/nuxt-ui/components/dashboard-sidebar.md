# DashboardSidebar

一个可调整大小且可折叠的侧边栏，用于仪表板中显示。

## 主要属性

- `resizable`：用于使侧边栏可调整大小。
- `collapsible`：用于在靠近屏幕边缘拖拽时使侧边栏可折叠。
- `side`：用于更改侧边栏的位置。
- `mode`：用于更改侧边栏菜单的模式。
- `toggle`：用于自定义在移动设备上显示的 [DashboardSidebarToggle](/docs/components/dashboard-sidebar-toggle) 组件。

## 使用方法

```vue
<UDashboardSidebar
  <!-- 在此处添加属性 -->
/>
```
