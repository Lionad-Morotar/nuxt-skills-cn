# InputDate

用于日期选择的输入组件。

> 基于 [Reka UI InputDate](https://reka-ui.com/docs/components/date-field)

## 主要属性

- `range`：用于选择日期范围。
- `color`：用于更改 InputDate 的颜色。
- `variant`：用于更改 InputDate 的变体。
- `size`：用于更改 InputDate 的尺寸。
- `disabled`：用于禁用 InputDate。

## 用法

```vue
<UInputDate
  <!-- 这里放置属性 -->
/>
```
