# usePreferredLanguages

响应式导航语言。它为 Web 开发者提供有关用户首选语言的信息。例如，这可能有助于根据用户的首选语言调整用户界面的语言，从而提供更好的用户体验。

**包：** `@vueuse/core`  
**类别：** 浏览器

## 用法

```ts
import { usePreferredLanguages } from '@vueuse/core'

const languages = usePreferredLanguages()
```
