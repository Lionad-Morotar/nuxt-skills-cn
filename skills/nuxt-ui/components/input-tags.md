# InputTags

一个用于显示交互式标签的输入组件。

> 基于 [Reka UI InputTags](https://reka-ui.com/docs/components/tags-input)

## 主要属性

- `placeholder`：用于设置占位符文本。
- `color`：用于在 InputTags 获得焦点时更改边框颜色。
- `variant`：用于更改 InputTags 的外观。
- `size`：用于调整 InputTags 的大小。
- `icon`：用于在 InputTags 内部显示 [Icon](/docs/components/icon)。
- `avatar`：用于在 InputTags 内部显示 [Avatar](/docs/components/avatar)。
- `loading`：用于在 InputTags 内部显示加载图标。
- `disabled`：用于禁用 InputTags。

## 使用方法

```vue
<UInputTags
  <!-- 属性在此处 -->
/>
```
