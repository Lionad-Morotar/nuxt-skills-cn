# useNProgress

用于 nprogress 的响应式封装。

**包：** `@vueuse/integrations`  
**类别：** '@Integrations'

## 返回值

| 名称      | 类型       |
| --------- | ---------- |
| isLoading | `computed` |
| progress  | `toRef`    |
| start     | `Ref`      |
| done      | `Ref`      |
| remove    | `Ref`      |
