# 按钮

一个可作为链接或触发操作的按钮元素。

## 主要属性

- `label`：用于设置按钮的标签。
- `color`：用于更改按钮的颜色。
- `variant`：用于更改按钮的变体。
- `size`：用于更改按钮的尺寸。
- `icon`：用于在按钮内显示 [Icon](/docs/components/icon)。
- `avatar`：用于在按钮内显示 [Avatar](/docs/components/avatar)。
- `loading`：用于显示加载图标并禁用按钮。
- `disabled`：用于禁用按钮。
- `class`：用于覆盖按钮的基础样式。
- `ui`：用于覆盖按钮的插槽样式。

## 用法

```vue
<UButton
  <!-- 属性在此处 -->
/>
```
