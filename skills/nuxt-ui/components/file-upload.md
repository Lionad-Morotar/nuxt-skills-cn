# 文件上传

一个用于上传文件的输入组件。

## 主要属性

- `multiple`：允许选择多个文件。
- `dropzone`：启用或禁用拖拽区域。
- `interactive`：启用或禁用可点击区域。
- `accept`：指定输入允许的文件类型。
- `label`：设置文件上传组件的标签。
- `description`：设置文件上传组件的描述。
- `icon`：设置文件上传组件的图标。
- `color`：更改文件上传组件的颜色。
- `variant`：更改文件上传组件的变体。
- `size`：更改文件上传组件的尺寸。

## 使用方法

```vue
<UFileUpload
  <!-- 属性在此处 -->
/>
```
