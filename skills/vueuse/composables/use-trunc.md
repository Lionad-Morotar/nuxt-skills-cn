# useTrunc

响应式。

**包：** `@vueuse/math`  
**分类：** '@Math'

## 用法

```ts
import { useTrunc } from '@vueuse/math'

const value1 = ref(0.95)
const value2 = ref(-2.34)
const result1 = useTrunc(value1) // 0
const result2 = useTrunc(value2) // -2
```
