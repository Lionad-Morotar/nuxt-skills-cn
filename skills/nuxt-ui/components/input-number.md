# InputNumber

用于输入数值的组件，支持自定义取值范围。

## 主要属性

- `step`：设置 InputNumber 的步长值。
- `orientation`：更改 InputNumber 的方向。
- `placeholder`：设置占位符文本。
- `color`：当 InputNumber 获得焦点时更改环形颜色。
- `variant`：更改 InputNumber 的变体样式。
- `size`：更改 InputNumber 的尺寸大小。
- `disabled`：禁用 InputNumber。

## 使用方法

```vue
<UInputNumber
  <!-- 属性在此处 -->
/>
```
