# useFps

响应式 FPS（每秒帧数）。

**包：** `@vueuse/core`
**类别：** 传感器

## 用法

```ts
import { useFps } from '@vueuse/core'

const fps = useFps()
```

## 选项

| 选项 | 类型     | 默认值 | 描述                             |
| ---- | -------- | ------ | -------------------------------- |
| every  | `number` | 10     | 每隔 x 帧计算一次 FPS。         |
