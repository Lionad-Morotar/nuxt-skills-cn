# 滑块

用于在指定范围内选择数值的输入组件。

> 基于 [Reka UI 滑块](https://reka-ui.com/docs/components/slider)

## 主要属性

- `step`：设置滑块的增量值。
- `orientation`：更改滑块的方向。
- `color`：更改滑块的颜色。
- `size`：更改滑块的尺寸。
- `tooltip`：在滑块手柄周围显示带有当前值的 [Tooltip](/docs/components/tooltip)。
- `disabled`：禁用滑块。
- `inverted`：视觉上反转滑块。

## 使用方法

```vue
<USlider
  <!-- 属性在此处 -->
/>
```
