# PinInput

用于输入 PIN 码的输入组件。

> 基于 [Reka UI PinInput](https://reka-ui.com/docs/components/pin-input)

## 核心属性

- `type`：用于更改输入类型。
- `mask`：用于将输入视为密码。
- `otp`：用于启用一次性密码功能。
- `length`：用于更改输入项的数量。
- `placeholder`：用于设置占位符文本。
- `color`：用于在 PinInput 获得焦点时更改边框颜色。
- `variant`：用于更改 PinInput 的变体样式。
- `size`：用于更改 PinInput 的尺寸。
- `disabled`：用于禁用 PinInput。

## 使用方法

```vue
<UPinInput
  <!-- 这里放置属性 -->
/>
```
