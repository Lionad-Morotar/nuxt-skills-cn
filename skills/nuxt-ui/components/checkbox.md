# 复选框

一种输入元素，用于在选中和未选中状态之间切换。

> 基于 [Reka UI 复选框](https://reka-ui.com/docs/components/checkbox)

## 主要属性

- `label`：用于设置复选框的标签。
- `description`：用于设置复选框的描述。
- `icon`：用于设置复选框选中时的图标。
- `color`：用于更改复选框的颜色。
- `variant`：用于更改复选框的变体。
- `size`：用于更改复选框的尺寸。
- `indicator`：用于更改指示器的位置或隐藏指示器。
- `disabled`：用于禁用复选框。

## 用法

```vue
<UCheckbox
  <!-- 属性在此处 -->
/>
```
