# usePreferredReducedTransparency

响应式 prefers-reduced-transparency 媒体查询。

**包：** `@vueuse/core`
**分类：** 浏览器

## 用法

```ts
import { usePreferredReducedTransparency } from '@vueuse/core'

const preferredTransparency = usePreferredReducedTransparency()
```
