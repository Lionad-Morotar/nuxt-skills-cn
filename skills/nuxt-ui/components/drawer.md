# 抽屉

一个可流畅滑入和滑出屏幕的抽屉组件。

## 主要属性

- `title`：用于设置抽屉头部的标题。
- `description`：用于设置抽屉头部的描述。
- `direction`：用于控制抽屉的滑出方向。
- `inset`：用于将抽屉从边缘向内偏移。
- `handle`：用于控制抽屉是否显示手柄。
- `overlay`：用于控制抽屉是否显示遮罩层。
- `modal`：用于控制抽屉是否阻止外部内容的交互。
- `dismissible`：用于控制点击抽屉外部或按下 Esc 键是否可关闭抽屉。

## 使用方式

```vue
<UDrawer
  <!-- 属性在此处 -->
/>
```

## 插槽

- `#content`
- `#header`
- `#body`
- `#footer`
