# useDevicePixelRatio

响应式跟踪

**包：** `@vueuse/core`
**分类：** 传感器

## 用法

```ts
import { useDevicePixelRatio } from '@vueuse/core'

const { pixelRatio } = useDevicePixelRatio()
```

## 返回值

| 名称       | 类型         |
| ---------- | ------------ |
| pixelRatio | `shallowRef` |
| stop       | `Ref`        |
