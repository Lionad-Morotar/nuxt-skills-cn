# useObservable

使用 RxJS 的 observable，返回一个值，并在组件卸载时自动取消订阅。

**包：** `@vueuse/rxjs`  
**类别：** '@RxJS'

## 选项

| 选项           | 类型             | 默认值  | 描述                                                         |
| -------------- | ---------------- | ------- | ------------------------------------------------------------ |
| initialValue   | `I \| undefined` | -       | 如果 observable 尚未发出值，则应设置的值。                   |

[VueUse 文档](https://vueuse.org/core/useObservable/)
