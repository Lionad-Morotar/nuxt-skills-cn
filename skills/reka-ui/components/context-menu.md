# 上下文菜单

## ContextMenuRoot

### 属性

| 属性    | 类型             | 默认值 |
| ------- | ---------------- | ------ |
| `dir`   | `"ltr" \| "rtl"` | -      |
| `modal` | `boolean`        | `true` |

### 事件

| 事件          | 参数                 |
| ------------- | -------------------- |
| `update:open` | `[payload: boolean]` |
