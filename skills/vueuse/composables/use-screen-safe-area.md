# useScreenSafeArea

响应式

**包：** `@vueuse/core`
**类别：** 浏览器

## 用法

```ts
import { useScreenSafeArea } from '@vueuse/core'

const {
  top,
  right,
  bottom,
  left,
} = useScreenSafeArea()
```

## 返回值

| 名称   | 类型         |
| ------ | ------------ |
| top    | `shallowRef` |
| right  | `shallowRef` |
| bottom | `shallowRef` |
| left   | `shallowRef` |
| update | `Ref`        |
