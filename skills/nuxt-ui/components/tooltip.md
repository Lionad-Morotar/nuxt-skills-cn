# 工具提示

一种弹出式窗口，在将鼠标悬停在元素上时显示相关信息。

> 基于 [Reka UI 工具提示](https://reka-ui.com/docs/components/tooltip)

## 核心属性

- `text`：用于设置工具提示的内容。
- `kbds`：用于在工具提示中渲染 [Kbd](/docs/components/kbd) 组件。
- `content`：用于控制工具提示内容的渲染方式，例如其 `align` 或 `side` 等。
- `arrow`：用于在工具提示上显示箭头。
- `disabled`：用于禁用工具提示。

## 使用方法

```vue
<UTooltip
  <!-- 属性在此处 -->
/>
```
